# 🛒 ShopSphere - Full Stack E-Commerce Website

A professional Amazon/Flipkart-style full stack e-commerce platform built using modern web technologies.

---

## 🚀 Features

### 👤 User Features

- User Registration & Login
- JWT Authentication
- Product Browsing
- Product Search & Filter
- Product Details
- Add to Cart
- Wishlist
- Checkout
- Order History
- Profile Management


### 👨‍💼 Admin Features

- Admin Dashboard
- Manage Products
- Manage Users
- Manage Orders
- Update Order Status
- Analytics Dashboard
- Revenue Tracking


### 💳 Payment Features

- Razorpay Payment Integration
- Payment Status Tracking


### 🖼️ Media Features

- Cloudinary Image Upload


### 📧 Communication

- Email Notifications

---
